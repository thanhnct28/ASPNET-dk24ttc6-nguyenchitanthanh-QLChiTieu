# BÁO CÁO TIẾN ĐỘ TUẦN 4
**Thời gian:** 29/03/2026 - 04/04/2026

## 1. Nội dung đã làm

- Hoàn thiện chức năng quản lý chi tiêu (ChiTieu.aspx)

  - Giao diện nhập:
    - Số tiền  
    - Ngày chi  
    - Danh mục (DropDownList)  
    - Ghi chú  

  - Chỉnh lại bố cục:
    - Dùng Bootstrap (grid)  
    - Card layout  
    - Nút lưu nhìn rõ hơn  

- Hiển thị danh sách chi tiêu:

  - Dùng GridView  
  - Các cột:
    - Mã  
    - Tên danh mục  
    - Số tiền  
    - Ngày  
    - Ghi chú  

  - Sắp xếp theo ngày giảm dần  
  - Format hiển thị:
    - Số tiền có dấu (vd: 1.000.000)  
    - Ngày dạng dd/MM/yyyy  

- Chức năng CRUD:

  - Thêm:
    - Nhập từ form  
    - Kiểm tra dữ liệu  
    - Insert vào database  

  - Sửa:
    - Dùng edit của GridView  
    - Sửa trực tiếp trên bảng  
    - Update lại dữ liệu  

  - Xóa:
    - Dùng delete của GridView  
    - Có confirm bằng JavaScript  
    - Xóa khỏi database  

- Xử lý bằng ADO.NET:
  - SqlConnection  
  - SqlCommand  
  - SqlDataAdapter  
  - Có dùng Parameters để tránh lỗi SQL  

- Validate dữ liệu:

  - Số tiền:
    - Phải là số  
    - Lớn hơn 0  

  - Ngày:
    - Đúng định dạng  

- Giao diện + xử lý thêm:

  - Format tiền khi nhập bằng JavaScript  
  - Có thông báo khi:
    - Thêm  
    - Sửa  
    - Xóa  

- Kiểm tra đăng nhập:

  - Dùng `Session["user"]`  
  - Chỉ load dữ liệu của user hiện tại  
  - Chưa login thì không cho vào

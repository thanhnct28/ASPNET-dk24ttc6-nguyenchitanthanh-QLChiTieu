# BÁO CÁO TIẾN ĐỘ TUẦN 1
**Thời gian:** 08/03/2026 - 14/03/2026

## 1. Những việc đã làm trong tuần

Tuần đầu bắt đầu triển khai đề tài quản lý chi tiêu cá nhân.

- Tạo repository GitHub để lưu source code:  
  `ASPNET-dk24ttc6-nguyenchitanthanh-QLChiTieu`

- Tạo project ASP.NET WebForms bằng Visual Studio  
- Chia layout cơ bản bằng MasterPage (Site.Master)

- Tạo các trang chính:
  - DangNhap.aspx  
  - ChiTieu.aspx  

- Thiết kế database SQL Server gồm:
  - NguoiDung (lưu tài khoản)  
  - DanhMucChi (các loại chi tiêu)  
  - ChiTieu (thông tin chi tiêu)  
  - Dự định thêm bảng ThuNhap ở các tuần sau  

- Cấu hình kết nối database trong Web.config (ConnectionString), test kết nối OK

- Trang ChiTieu:
  - Nhập dữ liệu:
    - Số tiền  
    - Ngày chi  
    - Danh mục  
    - Ghi chú  

  - Hiển thị danh sách bằng GridView:
    - Các cột: mã, danh mục, số tiền, ngày, ghi chú  
    - Sắp xếp theo ngày chi mới nhất  

- Chức năng đã làm:
  - Thêm mới  
  - Sửa  
  - Xóa  

- Xử lý dữ liệu bằng ADO.NET:
  - SqlConnection  
  - SqlCommand  
  - SqlDataAdapter  

- Dùng Session để kiểm tra đăng nhập (chưa login thì chuyển về trang login)

- Kiểm tra dữ liệu nhập:
  - Số tiền phải là số  
  - Ngày đúng định dạng  

- Giao diện:
  - Dùng Bootstrap  
  - Thêm CSS  
  - Dùng icon Font Awesome  

- JavaScript:
  - Format số tiền khi nhập  
  - Hiện confirm khi xóa
